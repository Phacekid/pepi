## PEPI BOT

This is a telegram bot that displays recently minted cashtoken nfts for a particular Project e.g. [PEPI](https://pepi.cash/)

#### Setup

1. To install packages
   `npm install`
2. Setup environment variales using the _.env.example_ to create _.env_
3. You can customize based on your projects information, the _config.js_ file.

#### To start app

1. Production
   `npm start`
   or
2. Test
   `npm test`

#### Commands

`/minted` - can be used to display total nft minted in group
